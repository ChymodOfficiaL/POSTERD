## üì¶ Installation

To install the package, simply run the following command in your terminal:

```
$ pkg update -y && pkg upgrade -y
$ pkg install git
$ git clone https://github.com/ChymodOfficiaL/POSTERD
$ cd "POSTERD"
$ pip install -r requirements.txt
$ python run.py
```
